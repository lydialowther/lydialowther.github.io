Agent20

Agent20 is a hotel booking website for sports teams. Team representatives can post their lodging needs and then hotels in the area can submit special offers for the representative to choose from.

A bit of context

A prototype had already been created for Agent20 so we need to create something that met the needs of the client and user, but we were also able to reuse styles from the prototype.

We wanted the users to be able to easily select what they needed, and to choose what offer fit their needs best. I created the mockups and did the html and css for the project.
